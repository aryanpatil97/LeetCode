# 1295. Find Numbers with Even Number of Digits

Given an array nums, return how many numbers contain an even number of digits.

**Example 1:**
```
Input: nums = [12,345,2,6,7896]
Output: 2
```

**Constraints:**
- 1 <= nums.length <= 500
- 1 <= nums[i] <= 10^5

**Approach:**
- Count digits for each number, increment if even.
