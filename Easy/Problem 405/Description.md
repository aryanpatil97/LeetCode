# 405. Convert a Number to Hexadecimal

Given an integer num, return its hexadecimal string representation.

**Example 1:**
```
Input: num = 26
Output: "1a"
```

**Constraints:**
- -2^31 <= num <= 2^31 - 1

**Approach:**
- Use bitwise operations and a character array to build the hexadecimal string.
