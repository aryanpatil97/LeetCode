# 1108. Defanging an IP Address

Given a valid IP address, return a defanged version where every period "." is replaced by "[.]".

**Example 1:**
```
Input: address = "1.1.1.1"
Output: "1[.]1[.]1[.]1"
```

**Constraints:**
- The given address is a valid IPv4 address.

**Approach:**
- Use string replace method.
