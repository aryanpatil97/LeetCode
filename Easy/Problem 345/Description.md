# 345. Reverse Vowels of a String

Given a string s, reverse only the vowels of the string and return it.

**Example 1:**
```
Input: s = "hello"
Output: "holle"
```

**Constraints:**
- 1 <= s.length <= 3 * 10^5
- s consists of printable ASCII characters.

**Approach:**
- Use two pointers from both ends, swap vowels when found.
