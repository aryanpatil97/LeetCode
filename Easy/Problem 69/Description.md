# 69. Sqrt(x)

Given a non-negative integer `x`, compute and return the square root of `x`. Only return the integer part (truncate the decimal part).

**Example 1:**
```
Input: x = 8
Output: 2
```

**Constraints:**
- 0 <= x <= 2^31 - 1

**Approach:**
- Use binary search to find the integer square root.
